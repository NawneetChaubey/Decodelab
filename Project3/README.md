# 🎓 SmartCourse Recommender AI

An AI-powered course recommendation system built using **Python**, **Streamlit**, and **TF-IDF with Cosine Similarity**.

This project was developed as **Project 3** of the **DecodeLabs Artificial Intelligence Internship Program**.

---

## 📌 Project Overview

SmartCourse Recommender AI recommends the most relevant courses based on a user's selected interests. The recommendation engine uses **Natural Language Processing (NLP)** techniques with **TF-IDF Vectorization** and **Cosine Similarity** to match user preferences with course descriptions.

---

## 🚀 Features

* 🎯 AI-Based Course Recommendation
* 🔍 TF-IDF Vectorization
* 📊 Cosine Similarity Matching
* 📚 Interactive Streamlit Interface
* 🎓 Difficulty Filter
* 📈 Match Percentage
* 📄 Dataset Preview
* 📊 Category Distribution Chart
* ⚡ Fast and Lightweight

---

## 🧠 Recommendation Workflow

1. Load Course Dataset
2. Combine Course Information
3. Apply TF-IDF Vectorization
4. Calculate Cosine Similarity
5. Match User Interests
6. Display Top Course Recommendations

## ▶️ Installation

### Clone the repository

```bash
git clone https://github.com/NawneetChaubey/AI-Recommendation-System.git
```

### Move into the project folder

```bash
cd AI-Recommendation-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
