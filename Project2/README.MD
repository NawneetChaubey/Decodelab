# 🌸 FlowerSense AI

A simple and interactive **Iris Flower Classification** web application built using **Python**, **Streamlit**, and the **K-Nearest Neighbors (KNN)** machine learning algorithm.

This project was developed as **Project 2** of the **DecodeLabs Artificial Intelligence Internship Program**.

Live demo - https://flowersense-ai.streamlit.app/
---

## 📌 Project Overview

FlowerSense AI is a supervised machine learning application that predicts the species of an Iris flower based on four flower measurements.

The model is trained using the famous **Iris Dataset** from Scikit-learn and uses the **K-Nearest Neighbors (KNN)** classification algorithm.

---

## 🚀 Features

* 🌸 Predict Iris Flower Species
* 📊 Model Accuracy
* 📈 F1 Score
* 🔥 Confusion Matrix Visualization
* 📋 Dataset Preview
* 🎨 Interactive Streamlit User Interface
* ⚡ Fast and Lightweight

---

## 🧠 Machine Learning Workflow

1. Load Iris Dataset
2. Split Dataset into Training & Testing Sets
3. Apply Feature Scaling using StandardScaler
4. Train K-Nearest Neighbors (KNN) Model
5. Predict Flower Species
6. Evaluate Model Performance

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/FlowerSense-AI.git
```

Move into the project folder

```bash
cd FlowerSense-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
