# 📌 Projects

## 🤖 Project 1 — Rule-Based AI Chatbot

A simple chatbot developed using Python and Streamlit that responds to predefined commands.

### Features

- Rule-Based Conversation
- Interactive Chat Interface
- Command Recognition
- Streamlit UI

**Tech Stack**

- Python
- Streamlit

---

## 🌸 Project 2 — FlowerSense AI

A Machine Learning application that classifies Iris flower species using the K-Nearest Neighbors (KNN) algorithm.

### Features

- Iris Dataset Classification
- Interactive Prediction
- Model Accuracy
- Confusion Matrix
- Dataset Preview

**Tech Stack**

- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

---

## 🎓 Project 3 — AI Course Recommendation System

An intelligent recommendation system that suggests courses based on user interests and difficulty level using cosine similarity.

### Features

- Course Recommendation
- Similarity Matching
- Difficulty Filter
- Recommendation Score
- Interactive Dashboard

**Tech Stack**

- Python
- Streamlit
- Pandas
- Scikit-learn

---

## 📄 Project 4 — AI Image Text Recognition

An OCR application that extracts text from images using Computer Vision and EasyOCR.

### Features

- Image Upload
- OCR Text Extraction
- Confidence Score
- Image Preprocessing
- Download Extracted Text

**Tech Stack**

- Python
- Streamlit
- OpenCV
- EasyOCR
- Pillow

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/NawneetChaubey/Decodelab.git
```

Move into the repository

```bash
cd Decodelab
```

Open any project folder

```bash
cd Project1
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app

```bash
streamlit run app.py
```
